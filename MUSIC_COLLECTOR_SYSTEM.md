# 🎵 Sistema Automatizado de Coleta de Músicas Católicas\n\n## Visão Geral\n\nO **Sistema de Coleta de Músicas Católicas** é um coletor inteligente que busca, valida, classifica e ingere automaticamente músicas litúrgicas no banco de dados do CELEBRA V4.\n\n---\n\n## 🎯 Objetivo\n\nAbastecer continuamente o banco de dados com músicas católicas reais, estruturadas e prontas para uso em:\n- Sugestões automáticas de músicas para missa\n- Montagem de celebrações\n- Playlist litúrgica\n- Recomendações inteligentes\n\n---\n\n## 📦 Componentes\n\n### 1. Script Python (`music-collector.py`)\n\n**Responsabilidade:** Executar o pipeline completo de coleta\n\n**Passos:**\n1. **PASSO 1: Coleta** - Extrai listas de músicas de fontes confiáveis\n2. **PASSO 2: Enriquecimento** - Adiciona letra, cifra, vídeo, áudio\n3. **PASSO 3: Classificação** - Categoriza automaticamente\n4. **PASSO 4: Validação** - Remove duplicatas e valida qualidade\n5. **PASSO 5: Ingestão** - Salva no banco CELEBRA\n\n**Execução:**\n```bash\ncd /home/ubuntu/celebra-project\npython3 scripts/music-collector.py\n```\n\n**Saída:**\n- Arquivo JSON: `/home/ubuntu/celebra-project/data/musicas_coletadas.json`\n- Log de execução com estatísticas completas\n\n---\n\n### 2. Router tRPC (`music-collector.router.ts`)\n\n**Responsabilidade:** Expor funcionalidades via API\n\n**Endpoints:**\n\n#### `executarColeta` (Admin)\n```typescript\ntrpc.musicCollector.executarColeta.mutate({\n  limite: 100,\n  fontes: ['musicanaliturgia', 'portalkairos', 'cnbb']\n})\n```\n\n#### `listarMusicas` (Público)\n```typescript\ntrpc.musicCollector.listarMusicas.useQuery({\n  tipo: 'liturgica',\n  momento: 'entrada',\n  tempoLiturgico: 'comum',\n  limite: 50,\n  pagina: 1\n})\n```\n\n#### `obterEstatisticas` (Público)\n```typescript\ntrpc.musicCollector.obterEstatisticas.useQuery()\n// Retorna: { total, porTipo, porMomento, porTempo, porFonte }\n```\n\n#### `obterSugestoesParaMissa` (Público)\n```typescript\ntrpc.musicCollector.obterSugestoesParaMissa.useQuery({\n  tempoLiturgico: 'comum',\n  ano: 'A',\n  quantidade: 3\n})\n```\n\n#### `buscar` (Público)\n```typescript\ntrpc.musicCollector.buscar.useQuery({\n  termo: 'Ave Maria',\n  limite: 20\n})\n```\n\n#### `executarColetaAutomatica` (Admin)\n```typescript\ntrpc.musicCollector.executarColetaAutomatica.mutate()\n```\n\n#### `obterLogExecucao` (Admin)\n```typescript\ntrpc.musicCollector.obterLogExecucao.useQuery({ limite: 100 })\n```\n\n---\n\n### 3. Scheduler (`music-collector-scheduler.service.ts`)\n\n**Responsabilidade:** Executar coleta automática em intervalos regulares\n\n**Configuração:**\n- **Frequência:** Toda segunda-feira às 2:00 AM\n- **Cron:** `0 2 * * 1`\n\n**Inicialização:**\n```typescript\nimport { inicializarMusicCollectorScheduler } from './services/music-collector-scheduler.service';\n\n// No server startup\nawait inicializarMusicCollectorScheduler();\n```\n\n**Funcionalidades:**\n- Coleta automática de múltiplas fontes\n- Validação e remoção de duplicatas\n- Log de execução\n- Tratamento de erros\n\n---\n\n### 4. Dashboard React (`MusicCollectorDashboard.tsx`)\n\n**Responsabilidade:** Interface para gerenciar coleta\n\n**Features:**\n- 📊 Estatísticas em tempo real\n- 🎵 Listagem de músicas coletadas\n- 🔍 Busca por título/artista\n- 🏷️ Filtros por tipo, momento, tempo litúrgico\n- 📈 Gráficos de distribuição\n- ▶️ Botões para executar coleta manual\n\n**Acesso:**\n```\nhttps://seu-app.com/music-collector-dashboard\n```\n\n---\n\n## 📊 Estrutura de Dados\n\n### Música Coletada\n\n```json\n{\n  \"id\": \"uuid\",\n  \"titulo\": \"É um prazer, Senhor\",\n  \"artista\": \"Hinário CNBB\",\n  \"tipo\": \"liturgica\",\n  \"momento\": \"entrada\",\n  \"tempoLiturgico\": \"comum\",\n  \"letra\": \"[Letra completa]\",\n  \"cifra\": \"[Cifra em formato texto]\",\n  \"youtubeUrl\": \"https://youtube.com/...\",\n  \"audioUrl\": \"https://pixabay.com/...\",\n  \"fonte\": \"musicanaliturgia\",\n  \"licenca\": \"uso_liturgico\",\n  \"idioma\": \"pt-BR\",\n  \"tags\": [\"liturgica\", \"entrada\", \"comum\"],\n  \"created_at\": \"2026-04-21T08:00:00Z\"\n}\n```\n\n### Classificações Obrigatórias\n\n**Tipo:**\n- `liturgica` -
