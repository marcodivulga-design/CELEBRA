# 🎵 CELEBRA STUDIO - Módulo de Produção Musical\n\n## Visão Geral\n\n**CELEBRA STUDIO** é um módulo integrado ao CELEBRA que transforma a plataforma de um simples organizador litúrgico em uma **plataforma completa de produção + execução musical**.\n\n### Transformação\n\n```\nAntes: App de organização litúrgica\nDepois: Plataforma completa de produção + execução musical da missa\n```\n\n## 🎯 Funcionalidades Principais\n\n### 1. **Criar Arranjos**\n- Selecionar música do catálogo\n- Escolher estilo (acústico, worship, coral, orquestral, litúrgico, instrumental)\n- Configurar intensidade, vozes e instrumentos\n- IA gera múltiplas versões do arranjo\n\n### 2. **Separação de Stems**\n- Separação automática em componentes:\n  - Vozes\n  - Bateria\n  - Baixo\n  - Teclado\n  - Guitarras\n  - Cordas\n  - Vento\n  - Percussão\n\n### 3. **Mixer Multitrack**\n- Controle de volume por stem\n- Mute/Solo individual\n- Pan (esquerda/direita)\n- Efeitos (reverb, delay, compressor, equalizador)\n- Visualização de waveform\n- Player integrado\n\n### 4. **Exportação**\n- Mix final (MP3, WAV, FLAC)\n- Stems individuais\n- Presets de mix salvos\n- Qualidade configurável\n\n### 5. **Integração com CELEBRA**\n- Versões associadas à música original\n- Seleção de versão ao montar missa\n- Uso de stems durante celebração\n- Controle em tempo real\n\n## 🏗️ Arquitetura\n\n### Backend (tRPC Routers)\n\n#### `studioArrangements.router.ts`\n- `criarArranjo` - Criar novo arranjo com IA\n- `listarArranjos` - Listar arranjos de uma música\n- `gerarVersoes` - Gerar múltiplas versões\n- `separarStems` - Separação automática\n- `obterStems` - Obter stems de uma versão\n- `salvarPresetMix` - Salvar preset de mix\n- `listarPresetsMix` - Listar presets\n- `exportarMix` - Exportar mix final\n- `exportarStems` - Exportar stems\n- `salvarComoVersao` - Salvar como versão da música\n\n#### `studioMixer.router.ts`\n- `inicializarMixer` - Inicializar mixer com stems\n- `ajustarVolume` - Ajustar volume de stem\n- `mutar` - Mutar/desmutar\n- `solo` - Solo de stem\n- `ajustarPan` - Ajustar pan\n- `aplicarEfeito` - Aplicar efeitos\n- `obterEstado` - Obter estado atual\n- `ajustarVolumeMestre` - Volume mestre\n- `resetar` - Resetar mixer\n\n### Frontend (React Components)\n\n#### `CelebraStudio.tsx`\n- Interface principal com 3 abas:\n  1. **Arranjos** - Criar e gerenciar arranjos\n  2. **Stems** - Separação e visualização\n  3. **Mixer** - Controle de reprodução\n\n## 📱 Fluxo de Uso\n\n### Passo 1: Selecionar Música\n```\nDashboard → CELEBRA STUDIO → Selecionar Música\n```\n\n### Passo 2: Criar Arranjo\n```\nEscolher estilo → Configurar intensidade → IA gera versões\n```\n\n### Passo 3: Separar Stems\n```\nSelecionar versão → Separar automático → Visualizar stems\n```\n\n### Passo 4: Mixer\n```\nAbrir mixer → Ajustar volumes → Aplicar efeitos → Reproduzir\n```\n\n### Passo 5: Exportar\n```\nSalvar preset → Exportar mix → Usar em celebração\n```\n\n## 🔗 Integração com CELEBRA\n\n### Estrutura de Dados\n\n```typescript\n// Música com versões\nmusica {\n  id: string\n  titulo: string\n  artista: string\n  versaoOriginal: Audio\n  versoes: [\n    {\n      id: string\n      nome: string\n      estilo: string\n      audio: Audio\n      stems: Stem[]\n      presets: MixPreset[]\n    }\n  ]\n}\n\n// Stem\nstem {\n  tipo: 'vozes' | 'bateria' | 'baixo' | 'teclado' | ...\n  url: string\n  duracao: number\n}\n\n// Mix Preset\npreset {\n  id: string\n  nome: string\n  mix: {\n    [stemId]: {\n      volume: number\n      muted: boolean\n      solo: boolean\n      efeitos: {}\n    }\n  }\n}\n```\n\n### Integração em Celebração\n\n```typescript\n// Ao montar missa\ncelebracao {\n  musicas: [\n    {\n      id: string\n      musicaId: string\n      versaoId: string  // Versão do STUDIO\n      presetId?: string // Preset de mix\n      momento: 'entrada' | 'comunhao' | ...\n    }\n  ]\n}\n```\n\n## 🎛️ Modo Mobile\n\n### Online\n- Geração de arranjos com IA\n- Processamento de stems\n- Upload/download\n\n### Offline\n- Reprodução de áudios baixados\n- Controle de mixer local\n- Sincronização ao conectar\n\n## 🔐 públicos\n\n## 📊 Relatórios\n\n- Arranjos criados\n- Versões mais usadas\n- Stems mais utilizados\n- Presets populares\n- Uso em celebrações\n\n## 🚀 Roadmap\n\n### Fase 1 (Atual)\n- [x] Criar arranjos
